Provides a widget and shortcode that allows you to display your most recent Dribbble shots.

== Description ==

This plugin will enable a short code and widget that you can use to display your most recent Dribbble shots in any WordPress post, page, or sidebar. Max number of shots to display is 15.

To display within a post or page use the [zilla_dribbbler] short code:

'[zilla_dribbbler player="username" shots="5"]'

== Installation ==

1. Activate the plugin
2a. Add [zilla_dribbbler player="username" shots="5"] to any post or page
2b. Add the ZillaDribbbler widget to any sidebar area.
